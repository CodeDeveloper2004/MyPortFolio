Projects I worked on

E-commerce Website
The E-commerce Website for Medicine will provide a comprehensive, secure, and user-friendly platform for purchasing medicinal products. With features like prescription verification, secure payments, product categorization, and tracking, it offers convenience and safety to customers. Additionally, the admin panel allows for efficient management of the products and orders, ensuring smooth operations for the pharmacy. This project will contribute to the growing trend of online pharmacies, offering users a safe, reliable, and convenient method of purchasing medicine.

Chatting Application
This project is a real-time chat application built using Java AWT/Swing for the graphical user interface (GUI) and Java Socket Programming for networking. The application follows a client-server architecture where the server listens for incoming client connections, and multiple clients can communicate with each other in real-time.
Tic-Tac-Toe
The Tic-Tac-Toe game is a classic two-player game where players take turns marking cells on a 3x3 grid with either an "X" or "O". The goal is to be the first player to get three of their marks in a horizontal, vertical, or diagonal row.
Flappy Bird
The Flappy Bird Game is a popular arcade game that requires the player to control a bird, trying to navigate through a series of pipes without hitting them. The game features simple mechanics but challenges the player with increasing difficulty. This project recreates the iconic Flappy Bird game using Java with AWT (Abstract Window Toolkit) and Swing for the graphical user interface..
